== Description ==

After installing and activiating the plugin you will see a new widget in your widgets called Sticky Notes. You can also add Sticky Notes to posts and pages with a simple shortcode. eg [stickynotes title="My Title" url="http://www.google.com"]My content goes here[/stickynotes]
The plugin uses the google api to retrieve the font (http://fonts.googleapis.com/css?family=Reenie+Beanie:regular) to help make it look lovely.

== Installation ==

1. Install the plugin
2. Activate the plugin
3. Go to Appearance > Widgets
4. Add Sticky Notes Widget
5. Enter a title
6. Enter a url to link to (use # if you dont want t a link)
7. Click Save
8. Add the shortcode [stickynotes title="My Title" url="http://www.google.com"]My content goes here[/stickynotes] to any posts or pages you like
9. Don't forget to donate at http://www.sms-affiliate.co.uk/donate/ if you like the plugin.
